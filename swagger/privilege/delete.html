Allows an authenticated Owner user to delete a privilege that he (or she) is an associated owner of.

  1. Receive authentication token, ID of the privilege to update and update details (name, list of associated users)

  2. Validate authentication token with Cognito to get user email

  3. Get details of authenticated user from PostgresDB by email

  4. Check whether authenticated user is an owner of the privilege

  5. Get details of the privilege to delete from PostgresDB by ID

  6. Delete the requested privilege on zTeros

  7. Delete the requested privilege from PostgresDB
