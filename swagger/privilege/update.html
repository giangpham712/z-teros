Allows an authenticated Owner user to update a privilege that he (or she) is an associated owner of.

  1. Receive authentication token, ID of the privilege to update and update details (name, list of associated users)

  2. Validate authentication token with Cognito to get user email

  3. Get details of authenticated user from PostgresDB by email

  4. Get details of the privilege to update from PostgresDB by ID

  5. Get list of User users associated with requested privilege from zTeros

  6. Get list of Owner users associated with requested privilege from zTeros

  7. Check if authenticated user is an owner of the requested privilege

  8. Create link (if not yet exists) to the requested privilege for User users that should be associated User user (if any)

  9. Create link (if not yet exists) to the requested privilege for Owner users that should be associated Owner user (if any)

  10. Delete link (if exists) to the requested privilege for users that should no longer be associated User user (if any)

  11. Delete link (if exists) to the requested privilege for Owner users that should no longer be associated Owner user (if any)

  12. Update details (name) of the requested privilege into PostgresDB
