Allows an authenticated user to list all users (users and owners) associated with a privilege

  1. Receive authentication token and ID of requested privilege

  2. Validate authentication token with Cognito

  3. Get details of requested privilege (name, UID) from PostgresDB by ID

  4. Get list of User users associated with requested privilege from zTeros

  5. Get list of Owner users associated with requested privilege from zTeros

  6. Retrieve details (first name, last name, role, email) for associated owners from PostgresDB by UID

  7. Retrieve details (first name, last name, role, email) for associated users from PostgresDB by UID

  8. Return associated users (users and owners) with details
