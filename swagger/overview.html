This is a demonstration interface for the zTeros Encryption Service Interface.

It has been created to provide a demonstration of how the zTeros middleware can be used as part of a back-end API to support a file encryption user interface, and is intended to be used as a reference resource for how you could build zTeros into your project.


Technology Overview

  • The NestJS (A Typescript NodeJS framework) API handles all traffic

  • The Amazon Cognito User Authentication Service provides user authentication - visit https://aws.amazon.com/cognito/ for more.
    Note: With adaption, Microsoft Azure Active Directory, LDAP, Identity Server, Google OpenID or similar could be used as a replacement.

  • The PostgresDB is the master store of record data from zTeros and the associated AWS Cognito references.

Try It Out

Fork this repository to build your own test demonstrator. You will require a separate PostgresSQL DB and Amazon Cognito Instance. You will also need to contact GBR14 Ltd to request access to a demonstration instance of zTeros.
