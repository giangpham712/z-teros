Allows an administrator user to create a new user account.

  1. Receive user details (first name, last name, email)

  2. Create new Owner user on zTeros

  3. Create new User user on zTeros

  4. Create new user on PostgresDB

  5. Persist user details (first name, last name, email, zTeros UIDs) into the DB

  6. Return details of created user
