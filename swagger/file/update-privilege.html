Enabling an associated owner to update the access privileges for a file:

  1. Receive authentication token and ID of file to view and ID of privilege to set

  2. Validate authentication token with Cognito

  3. Get file details including privilege ID from PostgresDB by ID

  4. Get lists of associated users and owners of the new privilege from zTeros

  5. Check whether the current user is a user of the new privilege to set

  6. Assign the new privilege to the file

  7. Get content of the public face of the data record from zTeros

  8. Create private face for the data record using the new privilege on Zteros

  9. Update content of the private face of the data record on zTeros

  10. Update content of the public face of the data record to empty on zTeros
