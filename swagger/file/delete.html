The process for deleting a public facing or private record is the same, however the steps the API takes are different as follows:

Public

  1. Receive authentication token and ID of file to delete

  2. Validate authentication token with Cognito

  3. Get file details from PostgresDB by ID

  4. Delete data record from zTeros

  5. Delete file details from PostgresDB


Private

  1. Receive authentication token and ID of file to delete

  2. Validate authentication token with Cognito

  3. Get file details including privilege ID from PostgresDB by ID

  4. Get privilege details from PostgresDB

  5. Check whether the current user is a user of the privilege

  6. Delete private face of the data record from zTeros

  7. Delete public face of the data record from zTeros

  8. Delete file details from PostgresDB
