Allows an authenticated Owner user to create a new public record containing the contents of a file.

  1. Receive authentication token and file data (content, name, parent path, access (private or public))

  2. Validate authentication token with Cognito

  3. Check in PostgresDB if parent path exists

  4. Check in PostgresDB if file path already exists

  5. Encode file content to base64 string format

  6. Create new data record on zTeros

  7. Update content of public face of the data record on zTeros

  8. Persist details of new file into PostgresDB

  9. Return details of uploaded file
