Downloading either a public or private facing record is done in the same manner, requiring the relevant ID to be passed through to the API. There is an additional step that the API undertakes to download a private facing record however:

Public

  1. Receive authentication token and ID of file to download

  2. Validate authentication token with Cognito

  3. Get file details from PostgresDB by ID

  4. Get content of the public face of the data record from zTeros

  5. Decode content from base64 string format to buffer

  6. Return content of file


Private

  1. Receive authentication token and ID of file to download

  2. Validate authentication token with Cognito

  3. Get file details including privilege ID from PostgresDB by ID

  4. Get privilege details from PostgresDB

  5. Get content of the private face of the data record from zTeros

  6. Decode content from base64 string format to buffer

  7. Return content of file
