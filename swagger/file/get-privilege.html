Enabling an authenticated user to obtain the access privileges for a file:

  1. Receive authentication token and ID of file to view

  2. Validate authentication token with Cognito

  3. Get file details including privilege ID from PostgresDB by ID

  4. Get privilege details from PostgresDB

  5. Get lists of associated users and owners for privilege from zTeros

  6. Get details of associated users and owners from PostgresDB

  7. Return details of privilege
