Allows an administrator user to delete a user. Read more about administrator privileges in the zTeros documentation.

  1. Receive authentication token and ID of user to delete

  2. Validate authentication token with Cognito to get username (email)

  3. Get details of authenticated user from PostgresDB by username

  4. Check if authenticated user is an Admin user

  5. Get details of user to delete from PostgresDB by ID

  6. Delete Owner user from zTeros

  7. Delete User user from zTeros

  8. Delete user from Cognito

  9. Delete user from PostgresDB
